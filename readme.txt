For Energia,
1. Download Energia from - http://energia.nu/download/
2. Install Energia.
3. Download nRF Library from https://github.com/spirilis/Enrf24
4. Copy this Library in /energia/library folder.
5. After opening Energia, go to Tools > Boards > Boards Manager > Install TIVA C Launchpad.
6. Write your code now in the .ino file and upload using the upload button.


For Android Studio,
1. Download Android Studio - developer.android.com/android/studio‎
2. Install Android Studio.
3. Click on “New Project”
4. Build .apk file and test on your android device.
